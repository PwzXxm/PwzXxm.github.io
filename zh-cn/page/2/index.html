USACO Party Lamps

发布于 Aug 31 2016

有 \(N\) \((10 \le N \le 100)\) 盏亮瞎眼的灯，从 1 到 N 编号。

有四种按钮：

按钮用途
1反转所有的灯（开变为关，关变为开）
2反转编号为奇数的灯（如 1，3，5）
3反转编号为偶数的灯（如 2，4，6）
4反转编号为 \(3k+1 ; 当 ; k \ge 0\) 的灯 （如 1，4，7）

给出 \(C\)（按下按钮的次数, \(0 \le C \le 10000\)）和一些灯最后的状态，找出所有组不同的灯的状态。
